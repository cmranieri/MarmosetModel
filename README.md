## Docker

To build the Docker container:

```
cd docker

bash build-docker.sh
```

To run the Docker container:


```
bash run-docker.sh
```

If running for the first time, compile the ```.mod``` files by running:

```
nrnivmodl
```

To run a demo of the marmoset model:
```
python MarmosetBG.py
```
